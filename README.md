# 📖 About

Machine Learning is powered by mathematics.

Algorithms like Linear Regression, Support Vector Machines, Neural Networks, PCA, and Gradient Descent are built upon concepts from Linear Algebra, Calculus, Probability, Statistics, and Optimization.

This repository documents my **90-day journey** to build a strong mathematical foundation for Machine Learning by learning one concept every day.

Instead of memorizing formulas, the goal is to understand:

- 📌 What the concept means
- 🧠 Why it works
- 🤖 How it is used in Machine Learning
- 📊 Practical examples
- 📝 Key takeaways

# 🎯 Goals
